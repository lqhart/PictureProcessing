## 图片处理
1. 画笔工具  
   可以在画板上或图片上进行画画  
   可以改变画笔的大小和颜色  
   可以回退一步或清空所有  
2. 增强  
   可以调整图片的亮度、对比度、饱和度、色温等
3. 文字  
   可以输入文字显示在画板或图片上  
   可以随意拖动文本  
   可以改变文本的颜色和大小  
4. 滤镜  
   提供多个滤镜可以对图片进行美化  
5. 裁剪  
   裁剪框可以随意拖动，可以随意改变大小
